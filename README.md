# league-ranker
A work in progress solution for ranking teams in League of Legends e-sports.


<h5>Frontend Commands</h5>

<pre>
<code>
  cd frontend
  npm i
  npm run dev
</code>
</pre>
  


<h5>Backend Commands</h5>

<pre>
<code>
  cd backend
  python manage.py runserver 8080
</code>
</pre>


<h5>Getting the Data</h5>

<pre>
<code>
  cd data_scripts
  python get_data.py
</code>
</pre>
